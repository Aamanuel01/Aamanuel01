# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any of my projects, please report it responsibly.

**Please do not open a public GitHub issue for security vulnerabilities.**

Instead, contact me directly:

- 📧 Open a [GitHub Security Advisory](https://github.com/Aamanuel01/Aamanuel01/security/advisories/new) for private disclosure
- Or reach out via GitHub: [@Aamanuel01](https://github.com/Aamanuel01)

I will acknowledge your report as soon as possible and work with you to understand and address the issue promptly.

## Supported Versions

Security updates are applied to the latest version of each project only.

- Latest: ✅ Yes
- Older: ❌ No
