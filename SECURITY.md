# Security Policy
## Reporting a Vulnerability
Please report security issues privately via a [Security Advisory](/Aamanuel01/Aamanuel01/security/advisories/new) or by reaching out to [@Aamanuel01](/Aamanuel01). Do not open public issues.
## Supported Versions
Security updates are applied to the latest version of each project only.
- **Latest**: ✅
- **Older**: ❌
